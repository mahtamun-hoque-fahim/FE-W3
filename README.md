# Recipe Finder

A React app to search recipes using the TheMealDB API. Built as part of the FlyRank.ai Frontend AI Engineering internship, Week 3 assignment.

## Features
- Search recipes by name
- View recipe grid with images
- Click a recipe to see full instructions in a modal
- Dark theme UI

## Tech Stack
- React (Vite)
- TheMealDB API (free, no key required)

## Running Locally
\`\`\`bash
npm install
npm run dev
\`\`\`
